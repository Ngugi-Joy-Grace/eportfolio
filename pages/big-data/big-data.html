> Deciphering Big Data

August 2023

Learning Outcomes

  • Understand big data concepts, tech, and management, focusing on security risks and challenges.
  • Analyse data wrangling issues and determine the best tools and methods for solutions.
  • Explore various data types and storage formats, understanding memory and storage needs.
  • Review data collection methods, ensuring data integrity, reliability, and readiness before processing.
  • Explore data presentation methods, assessing data readability and longevity, and delve into cloud services and APIs for data connectivity.
  • Study the core concepts of DBMS, focusing on database design and modelling.
  • Investigate the future of data use, understanding foundational machine learning concepts and their application in big data management.
